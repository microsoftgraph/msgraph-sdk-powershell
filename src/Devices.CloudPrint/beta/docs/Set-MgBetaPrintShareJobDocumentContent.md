# Set-MgBetaPrintShareJobDocumentContent

## SYNOPSIS
Update media content for the navigation property documents in print

## SYNTAX

### Set (Default)
```
Set-MgBetaPrintShareJobDocumentContent -PrintDocumentId <String> -PrinterShareId <String> -PrintJobId <String>
 -InFile <String> [-Data <Stream>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgBetaPrintShareJobDocumentContent -InputObject <IDevicesCloudPrintIdentity> -InFile <String>
 [-Data <Stream>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
