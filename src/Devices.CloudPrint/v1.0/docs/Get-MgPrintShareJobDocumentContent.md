# Get-MgPrintShareJobDocumentContent

## SYNOPSIS
Get media content for the navigation property documents from print

## SYNTAX

### Get (Default)
```
Get-MgPrintShareJobDocumentContent -PrintDocumentId <String> -PrinterShareId <String> -PrintJobId <String>
 -OutFile <String> [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru]
 [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgPrintShareJobDocumentContent -InputObject <IDevicesCloudPrintIdentity> -OutFile <String>
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [<CommonParameters>]
```
