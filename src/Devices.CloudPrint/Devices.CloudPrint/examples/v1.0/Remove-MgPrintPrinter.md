### Example 1: Using the Remove-MgPrintPrinter Cmdlet
```powershell
Import-Module Microsoft.Graph.Devices.CloudPrint
Remove-MgPrintPrinter -PrinterId $printerId
```
This example shows how to use the Remove-MgPrintPrinter Cmdlet.
