### Example 1: Using the Remove-MgBetaOauth2PermissionGrant Cmdlet

```powershell
Import-Module Microsoft.Graph.Beta.Identity.SignIns
Remove-MgBetaOauth2PermissionGrant -OAuth2PermissionGrantId $oAuth2PermissionGrantId
```
