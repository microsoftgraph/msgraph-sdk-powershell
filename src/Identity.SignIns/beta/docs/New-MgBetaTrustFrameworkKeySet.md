# New-MgBetaTrustFrameworkKeySet

## SYNOPSIS
Create a new trustFrameworkKeySet.
The ID of the trustFrameworkKeySet is expected in the create request; however, it can be modified by the service.
The modified ID will be available in the response and in the location header.

## SYNTAX

### CreateExpanded (Default)
```
New-MgBetaTrustFrameworkKeySet [-AdditionalProperties <Hashtable>] [-Id <String>]
 [-Keys <IMicrosoftGraphTrustFrameworkKey[]>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Create
```
New-MgBetaTrustFrameworkKeySet -BodyParameter <IMicrosoftGraphTrustFrameworkKeySet> [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Create a new trustFrameworkKeySet.
The ID of the trustFrameworkKeySet is expected in the create request; however, it can be modified by the service.
The modified ID will be available in the response and in the location header.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```powershell
Import-Module Microsoft.Graph.Beta.Identity.SignIns
$params = @{
	Id = "keyset1"
	Keys = @(
		@{
			K = "k-value"
			X5c = @(
				"x5c-value"
			)
			X5t = "x5t-value"
			Kty = "kty-value"
			Use = "use-value"
			Exp = 99
			Nbf = 99
			Kid = "kid-value"
			E = "e-value"
			N = "n-value"
			D = "d-value"
			P = "p-value"
			Q = "q-value"
			Dp = "dp-value"
			Dq = "dq-value"
			Qi = "qi-value"
		}
	)
}
New-MgBetaTrustFrameworkKeySet -BodyParameter $params
```



### -------------------------- EXAMPLE 2 --------------------------
```powershell
Import-Module Microsoft.Graph.Beta.Identity.SignIns
$params = @{
	Id = "keyset1"
}
New-MgBetaTrustFrameworkKeySet -BodyParameter $params
```
