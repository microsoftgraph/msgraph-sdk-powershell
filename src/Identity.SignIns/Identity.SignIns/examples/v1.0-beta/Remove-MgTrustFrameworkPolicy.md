### Example 1: Using the Remove-MgTrustFrameworkPolicy Cmdlet

```powershell
Import-Module Microsoft.Graph.Identity.SignIns
Remove-MgTrustFrameworkPolicy -TrustFrameworkPolicyId $trustFrameworkPolicyId
```
