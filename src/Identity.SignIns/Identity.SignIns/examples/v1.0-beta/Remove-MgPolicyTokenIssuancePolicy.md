### Example 1: Using the Remove-MgPolicyTokenIssuancePolicy Cmdlet

```powershell
Import-Module Microsoft.Graph.Identity.SignIns
Remove-MgPolicyTokenIssuancePolicy -TokenIssuancePolicyId $tokenIssuancePolicyId
```
