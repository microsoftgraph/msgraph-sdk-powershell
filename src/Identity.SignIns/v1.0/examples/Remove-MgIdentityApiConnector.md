### Example 1: Using the Remove-MgIdentityApiConnector Cmdlet
```powershell
Import-Module Microsoft.Graph.Identity.SignIns
Remove-MgIdentityApiConnector -IdentityApiConnectorId $identityApiConnectorId
```
