# Update-MgDeviceAppManagementMobileAppInstallSummary

## SYNOPSIS

```
Update-MgDeviceAppManagementMobileAppInstallSummary -MobileAppId <String> [-FailedDeviceCount <Int32>]
 [-FailedUserCount <Int32>] [-Id <String>] [-InstalledDeviceCount <Int32>] [-InstalledUserCount <Int32>]
 [-NotApplicableDeviceCount <Int32>] [-NotApplicableUserCount <Int32>] [-NotInstalledDeviceCount <Int32>]
 [-NotInstalledUserCount <Int32>] [-PendingInstallDeviceCount <Int32>] [-PendingInstallUserCount <Int32>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

```
Update-MgDeviceAppManagementMobileAppInstallSummary -MobileAppId <String>
 -BodyParameter <IMicrosoftGraphMobileAppInstallSummary> [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
