# Get-MgContactDirectReport

## SYNOPSIS
Get directReports from contacts

## SYNTAX

### List (Default)
```
Get-MgContactDirectReport -OrgContactId <String> [-Count] [-ExpandProperty <String[]>] [-Filter <String>]
 [-PageSize <Int32>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>] [-Sort <String[]>]
 [<CommonParameters>]
```

### Get
```
Get-MgContactDirectReport -DirectoryObjectId <String> -OrgContactId <String> [-ExpandProperty <String[]>]
 [-Property <String[]>] [<CommonParameters>]
```
