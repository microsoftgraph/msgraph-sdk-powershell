# Get-MgUserOnenoteResource

## SYNOPSIS
The image and other file resources in OneNote pages.
Getting a resources collection isn't supported, but you can get the binary content of a specific resource.
Read-only.
Nullable.

## SYNTAX

### Get (Default)
```
Get-MgUserOnenoteResource -OnenoteResourceId <String> -UserId <String> [-ExpandProperty <String[]>]
 [-Property <String[]>] [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgUserOnenoteResource -InputObject <INotesIdentity> [-ExpandProperty <String[]>] [-Property <String[]>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [<CommonParameters>]
```

## DESCRIPTION
The image and other file resources in OneNote pages.
Getting a resources collection isn't supported, but you can get the binary content of a specific resource.
Read-only.
Nullable.
