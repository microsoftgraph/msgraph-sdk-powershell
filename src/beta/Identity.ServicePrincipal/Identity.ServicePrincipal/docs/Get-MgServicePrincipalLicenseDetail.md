# Get-MgServicePrincipalLicenseDetail

## SYNOPSIS
Get licenseDetails from servicePrincipals

## SYNTAX

### List (Default)
```
Get-MgServicePrincipalLicenseDetail -ServicePrincipalId <String> [-Count] [-Expand <String[]>]
 [-Filter <String>] [-Orderby <String[]>] [-Search <String>] [-Select <String[]>] [-Skip <Int32>]
 [-Top <Int32>] [<CommonParameters>]
```

### Get
```
Get-MgServicePrincipalLicenseDetail -LicenseDetailsId <String> -ServicePrincipalId <String>
 [-Expand <String[]>] [-Select <String[]>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgServicePrincipalLicenseDetail -InputObject <IIdentityServicePrincipalIdentity> [-Expand <String[]>]
 [-Select <String[]>] [<CommonParameters>]
```
