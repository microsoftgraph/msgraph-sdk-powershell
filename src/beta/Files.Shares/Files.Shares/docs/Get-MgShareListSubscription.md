# Get-MgShareListSubscription

## SYNOPSIS
Get subscriptions from shares

## SYNTAX

### List (Default)
```
Get-MgShareListSubscription -SharedDriveItemId <String> [-Count] [-Expand <String[]>] [-Filter <String>]
 [-Orderby <String[]>] [-Search <String>] [-Select <String[]>] [-Skip <Int32>] [-Top <Int32>]
 [<CommonParameters>]
```

### Get
```
Get-MgShareListSubscription -SharedDriveItemId <String> -SubscriptionId <String> [-Expand <String[]>]
 [-Select <String[]>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgShareListSubscription -InputObject <IFilesSharesIdentity> [-Expand <String[]>] [-Select <String[]>]
 [<CommonParameters>]
```
