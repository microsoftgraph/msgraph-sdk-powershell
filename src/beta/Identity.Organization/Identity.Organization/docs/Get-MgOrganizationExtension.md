# Get-MgOrganizationExtension

## SYNOPSIS
Get extensions from organization

## SYNTAX

### List (Default)
```
Get-MgOrganizationExtension -OrganizationId <String> [-Count] [-Expand <String[]>] [-Filter <String>]
 [-Orderby <String[]>] [-Search <String>] [-Select <String[]>] [-Skip <Int32>] [-Top <Int32>]
 [<CommonParameters>]
```

### Get
```
Get-MgOrganizationExtension -ExtensionId <String> -OrganizationId <String> [-Expand <String[]>]
 [-Select <String[]>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgOrganizationExtension -InputObject <IIdentityOrganizationIdentity> [-Expand <String[]>]
 [-Select <String[]>] [<CommonParameters>]
```
