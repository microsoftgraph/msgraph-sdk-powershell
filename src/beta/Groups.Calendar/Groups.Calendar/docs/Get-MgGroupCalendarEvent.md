# Get-MgGroupCalendarEvent

## SYNOPSIS
Get events from groups

## SYNTAX

### List (Default)
```
Get-MgGroupCalendarEvent -GroupId <String> [-Count] [-Expand <String[]>] [-Filter <String>]
 [-Orderby <String[]>] [-Search <String>] [-Select <String[]>] [-Skip <Int32>] [-Top <Int32>]
 [<CommonParameters>]
```

### Get
```
Get-MgGroupCalendarEvent -EventId <String> -GroupId <String> [-Expand <String[]>] [-Select <String[]>]
 [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgGroupCalendarEvent -InputObject <IGroupsCalendarIdentity> [-Expand <String[]>] [-Select <String[]>]
 [<CommonParameters>]
```
