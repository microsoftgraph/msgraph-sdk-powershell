# Get-MgDriveBundle

## SYNOPSIS
Get bundles from drives

## SYNTAX

### List1 (Default)
```
Get-MgDriveBundle [-Count] [-Expand <String[]>] [-Filter <String>] [-Orderby <String[]>] [-Search <String>]
 [-Select <String[]>] [-Skip <Int32>] [-Top <Int32>] [<CommonParameters>]
```

### Get
```
Get-MgDriveBundle -DriveId <String> -DriveItemId <String> [-Expand <String[]>] [-Select <String[]>]
 [<CommonParameters>]
```

### Get1
```
Get-MgDriveBundle -DriveItemId <String> [-Expand <String[]>] [-Select <String[]>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgDriveBundle -InputObject <IFilesDrivesIdentity> [-Expand <String[]>] [-Select <String[]>]
 [<CommonParameters>]
```

### GetViaIdentity1
```
Get-MgDriveBundle -InputObject <IFilesDrivesIdentity> [-Expand <String[]>] [-Select <String[]>]
 [<CommonParameters>]
```

### List
```
Get-MgDriveBundle -DriveId <String> [-Count] [-Expand <String[]>] [-Filter <String>] [-Orderby <String[]>]
 [-Search <String>] [-Select <String[]>] [-Skip <Int32>] [-Top <Int32>] [<CommonParameters>]
```
