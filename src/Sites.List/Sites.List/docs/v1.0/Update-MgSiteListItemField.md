# Update-MgSiteListItemField

## SYNOPSIS
Update the navigation property fields in sites

## SYNTAX

### UpdateExpanded1 (Default)
```
Update-MgSiteListItemField -ListId <String> -ListItemId <String> -SiteId <String> [-Id <String>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Update1
```
Update-MgSiteListItemField -ListId <String> -ListItemId <String> -SiteId <String>
 -BodyParameter <IMicrosoftGraphFieldValueSet> [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
