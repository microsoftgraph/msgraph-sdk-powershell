# Get-MgShareListSubscription

## SYNOPSIS
Get subscriptions from shares

## SYNTAX

### List (Default)
```
Get-MgShareListSubscription -SharedDriveItemId <String> [-Count] [-ExpandProperty <String[]>]
 [-Filter <String>] [-PageSize <Int32>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>]
 [-Sort <String[]>] [-All] [<CommonParameters>]
```

### Get
```
Get-MgShareListSubscription -SharedDriveItemId <String> -SubscriptionId <String> [-ExpandProperty <String[]>]
 [-Property <String[]>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgShareListSubscription -InputObject <IFilesSharesIdentity> [-ExpandProperty <String[]>]
 [-Property <String[]>] [<CommonParameters>]
```
