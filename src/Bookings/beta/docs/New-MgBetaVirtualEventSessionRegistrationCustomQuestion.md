# New-MgBetaVirtualEventSessionRegistrationCustomQuestion

## SYNOPSIS
Create a custom registration question associated with a meetingRegistration object on behalf of the organizer.
This API is available in the following national cloud deployments.

## SYNTAX

### CreateExpanded (Default)
```
New-MgBetaVirtualEventSessionRegistrationCustomQuestion -VirtualEventId <String>
 -VirtualEventSessionId <String> [-AdditionalProperties <Hashtable>] [-AnswerInputType <String>]
 [-AnswerOptions <String[]>] [-DisplayName <String>] [-Id <String>] [-IsRequired] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### Create
```
New-MgBetaVirtualEventSessionRegistrationCustomQuestion -VirtualEventId <String>
 -VirtualEventSessionId <String> -BodyParameter <IMicrosoftGraphMeetingRegistrationQuestion> [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentity
```
New-MgBetaVirtualEventSessionRegistrationCustomQuestion -InputObject <IBookingsIdentity>
 -BodyParameter <IMicrosoftGraphMeetingRegistrationQuestion> [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentityExpanded
```
New-MgBetaVirtualEventSessionRegistrationCustomQuestion -InputObject <IBookingsIdentity>
 [-AdditionalProperties <Hashtable>] [-AnswerInputType <String>] [-AnswerOptions <String[]>]
 [-DisplayName <String>] [-Id <String>] [-IsRequired] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create a custom registration question associated with a meetingRegistration object on behalf of the organizer.
This API is available in the following national cloud deployments.
