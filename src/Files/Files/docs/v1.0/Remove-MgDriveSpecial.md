# Remove-MgDriveSpecial

## SYNOPSIS
Collection of common folders available in OneDrive.
Read-only.
Nullable.

## SYNTAX

### Delete1 (Default)
```
Remove-MgDriveSpecial -DriveId <String> -DriveItemId <String> [-IfMatch <String>] [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity1
```
Remove-MgDriveSpecial -InputObject <IFilesIdentity> [-IfMatch <String>] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Collection of common folders available in OneDrive.
Read-only.
Nullable.
