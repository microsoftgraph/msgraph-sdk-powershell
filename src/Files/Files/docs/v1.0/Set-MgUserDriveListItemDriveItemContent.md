# Set-MgUserDriveListItemDriveItemContent

## SYNOPSIS
Update content for the navigation property driveItem in users

## SYNTAX

### Set (Default)
```
Set-MgUserDriveListItemDriveItemContent -DriveId <String> -ListItemId <String> -UserId <String>
 -InFile <String> [-BodyParameter <Stream>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgUserDriveListItemDriveItemContent -InputObject <IFilesIdentity> -InFile <String>
 [-BodyParameter <Stream>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Update content for the navigation property driveItem in users
