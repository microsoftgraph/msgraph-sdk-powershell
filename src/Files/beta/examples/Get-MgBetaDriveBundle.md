### Example 1: Get a bundle

```powershell
Import-Module Microsoft.Graph.Beta.Files

Get-MgBetaDriveBundle -DriveId $driveId -DriveItemId $driveItemId
```
This example shows how to use the Get-MgBetaDriveBundle Cmdlet.
