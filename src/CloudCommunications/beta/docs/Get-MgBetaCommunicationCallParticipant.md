# Get-MgBetaCommunicationCallParticipant

## SYNOPSIS
Retrieve the properties and relationships of a participant object.

## SYNTAX

### List (Default)
```
Get-MgBetaCommunicationCallParticipant -CallId <String> [-ExpandProperty <String[]>] [-Filter <String>]
 [-Property <String[]>] [-Search <String>] [-Skip <Int32>] [-Sort <String[]>] [-Top <Int32>]
 [-ResponseHeadersVariable <String>] [-All] [-CountVariable <String>] [-Headers <IDictionary>]
 [-PageSize <Int32>] [<CommonParameters>]
```

### Get
```
Get-MgBetaCommunicationCallParticipant -CallId <String> -ParticipantId <String> [-ExpandProperty <String[]>]
 [-Property <String[]>] [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-MgBetaCommunicationCallParticipant -InputObject <ICloudCommunicationsIdentity>
 [-ExpandProperty <String[]>] [-Property <String[]>] [-ResponseHeadersVariable <String>]

```

## DESCRIPTION
Retrieve the properties and relationships of a participant object.
