### Example 1: Get an owner of a group

This example gets the owner of the specified group.

```powershell
Get-MgGroupOwner -GroupId '4d5f57a1-85e0-41dd-8282-ff995ad5e1c3'
```

```Output
Id                                   DeletedDateTime
--                                   ---------------
5fc5c052-8774-4258-8705-0b4ab3e9a2df
```
