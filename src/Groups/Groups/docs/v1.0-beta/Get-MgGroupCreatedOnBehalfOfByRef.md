# Get-MgGroupCreatedOnBehalfOfByRef

## SYNOPSIS
The user (or application) that created the group.
NOTE: This is not set if the user is an administrator.
Read-only.

```
Get-MgGroupCreatedOnBehalfOfByRef -GroupId <String> [<CommonParameters>]
```
