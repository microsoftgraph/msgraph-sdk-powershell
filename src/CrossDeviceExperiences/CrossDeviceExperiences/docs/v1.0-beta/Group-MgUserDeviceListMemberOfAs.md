# Group-MgUserDeviceListMemberOfAs

## SYNOPSIS

### Group (Default)
```
Group-MgUserDeviceListMemberOfAs -DeviceId <String> -UserId <String> [-Count] [-ExpandProperty <String[]>]
 [-Filter <String>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>] [-Sort <String[]>]
 [-Top <Int32>] [-ConsistencyLevel <String>] [<CommonParameters>]
```

```
Group-MgUserDeviceListMemberOfAs -InputObject <ICrossDeviceExperiencesIdentity> [-Count]
```
