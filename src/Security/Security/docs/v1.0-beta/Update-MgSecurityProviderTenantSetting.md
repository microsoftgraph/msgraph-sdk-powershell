```
Update-MgSecurityProviderTenantSetting -ProviderTenantSettingId <String> [-AzureTenantId <String>] [-Enabled]

 [-Confirm] [-WhatIf] [<CommonParameters>]
```

Update-MgSecurityProviderTenantSetting -ProviderTenantSettingId <String>
