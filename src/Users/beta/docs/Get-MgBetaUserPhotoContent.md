# Get-MgBetaUserPhotoContent

## SYNOPSIS

## SYNTAX

### Get (Default)
```
Get-MgBetaUserPhotoContent -UserId <String> -OutFile <String> [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [<CommonParameters>]
```
