### Example 1: Get a user's manager

The commands get the manager of the specified user.

```powershell
Get-MgUserManager -UserId "0ec3a5e8-b4b6-4678-90ff-ce786055065f"
```

```Output
Id                                   DeletedDateTime
--                                   ---------------
e1428c8c-4219-4beb-b5cb-765c2590278b
```
