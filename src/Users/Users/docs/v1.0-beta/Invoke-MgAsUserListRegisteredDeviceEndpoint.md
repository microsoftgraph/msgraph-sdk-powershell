# Invoke-MgAsUserListRegisteredDeviceEndpoint

## SYNOPSIS
Get the items of type microsoft.graph.endpoint in the microsoft.graph.directoryObject collection

## SYNTAX

### As (Default)
```
Invoke-MgAsUserListRegisteredDeviceEndpoint -UserId <String> [-Count] [-ExpandProperty <String[]>]
 [-Filter <String>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>] [-Sort <String[]>]
 [-Top <Int32>] [-ConsistencyLevel <String>] [<CommonParameters>]
```

### AsViaIdentity
```
Invoke-MgAsUserListRegisteredDeviceEndpoint -InputObject <IUsersIdentity> [-Count]
 [-ExpandProperty <String[]>] [-Filter <String>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>]
 [-Sort <String[]>] [-Top <Int32>] [-ConsistencyLevel <String>] [<CommonParameters>]
```

## DESCRIPTION
Get the items of type microsoft.graph.endpoint in the microsoft.graph.directoryObject collection
