# Get-MgUserAuthenticationPhoneMethod

## SYNOPSIS

### List (Default)
```
Get-MgUserAuthenticationPhoneMethod -UserId <String> [-Count] [-ExpandProperty <String[]>] [-Filter <String>]
 [-PageSize <Int32>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>] [-Sort <String[]>] [-All]
 [<CommonParameters>]
```
