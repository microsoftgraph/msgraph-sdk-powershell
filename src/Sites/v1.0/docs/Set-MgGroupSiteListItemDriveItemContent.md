# Set-MgGroupSiteListItemDriveItemContent

## SYNOPSIS
The content stream, if the item represents a file.

## SYNTAX

### Set (Default)
```
Set-MgGroupSiteListItemDriveItemContent -GroupId <String> -ListId <String> -ListItemId <String>
 -SiteId <String> -InFile <String> [-Data <Stream>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgGroupSiteListItemDriveItemContent -InputObject <ISitesIdentity> -InFile <String> [-Data <Stream>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
The content stream, if the item represents a file.
