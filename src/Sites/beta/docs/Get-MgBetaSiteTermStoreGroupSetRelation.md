# Get-MgBetaSiteTermStoreGroupSetRelation

## SYNOPSIS
Indicates which terms have been pinned or reused directly under the set.

## SYNTAX

### List (Default)
```
Get-MgBetaSiteTermStoreGroupSetRelation -GroupId <String> -SetId <String> -SiteId <String>
 [-ExpandProperty <String[]>] [-Filter <String>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>]
 [-Sort <String[]>] [-Top <Int32>] [-ResponseHeadersVariable <String>] [-All] [-CountVariable <String>]
 [-Headers <IDictionary>] [-PageSize <Int32>] [<CommonParameters>]
```

### Get

Get-MgBetaSiteTermStoreGroupSetRelation -GroupId <String> -RelationId <String> -SetId <String>
 -SiteId <String> [-ExpandProperty <String[]>] [-Property <String[]>] [-ResponseHeadersVariable <String>]
