# Get-MgUserActivity

## SYNOPSIS
Get activities from users

## SYNTAX

### List1 (Default)
```
Get-MgUserActivity -UserId <String> [-Count] [-ExpandProperty <String[]>] [-Filter <String>]
 [-PageSize <Int32>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>] [-Sort <String[]>]
 [<CommonParameters>]
```

### Get
```
Get-MgUserActivity -UserActivityId <String> -UserId <String> [-ExpandProperty <String[]>]
 [-Property <String[]>] [<CommonParameters>]
```
