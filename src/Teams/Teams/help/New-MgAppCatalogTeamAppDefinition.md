# New-MgAppCatalogTeamAppDefinition

## SYNOPSIS
Create new navigation property to appDefinitions for appCatalogs

## SYNTAX

### CreateExpanded1 (Default)
```
New-MgAppCatalogTeamAppDefinition -TeamsAppId <String> [-AdditionalProperties <Hashtable>]
 [-AzureAdAppId <String>] [-CreatedBy <IMicrosoftGraphIdentitySet>] [-Description <String>]
 [-DisplayName <String>] [-Id <String>] [-LastModifiedDateTime <DateTime>] [-PublishingState <String>]
 [-Shortdescription <String>] [-TeamsAppId1 <String>] [-Version <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### CreateViaIdentityExpanded1
```
New-MgAppCatalogTeamAppDefinition [-TeamsAppId <String>] -InputObject <ITeamsIdentity>
 [-AdditionalProperties <Hashtable>] [-AzureAdAppId <String>] [-CreatedBy <IMicrosoftGraphIdentitySet>]
 [-Description <String>] [-DisplayName <String>] [-Id <String>] [-LastModifiedDateTime <DateTime>]
 [-PublishingState <String>] [-Shortdescription <String>] [-Version <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Create1
```
New-MgAppCatalogTeamAppDefinition -TeamsAppId <String> -BodyParameter <IMicrosoftGraphTeamsAppDefinition>
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### CreateViaIdentity1
```

 -BodyParameter <IMicrosoftGraphTeamsAppDefinition> [-WhatIf] [-Confirm] [<CommonParameters>]
```
