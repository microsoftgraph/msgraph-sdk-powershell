# Remove-MgTeamChannel

## SYNOPSIS
Delete the channel.

## SYNTAX

### Delete (Default)
```
Remove-MgTeamChannel -ChannelId <String> -TeamId <String> [-IfMatch <String>]
 [-ResponseHeadersVariable <String>] [-Headers <IDictionary>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-MgTeamChannel -InputObject <ITeamsIdentity> [-IfMatch <String>] [-ResponseHeadersVariable <String>]
 [-Headers <IDictionary>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Delete the channel.

## EXAMPLES

Import-Module Microsoft.Graph.Teams
