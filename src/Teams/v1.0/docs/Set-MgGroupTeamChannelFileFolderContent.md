# Set-MgGroupTeamChannelFileFolderContent

## SYNOPSIS
The content stream, if the item represents a file.

## SYNTAX

### Set (Default)
```
Set-MgGroupTeamChannelFileFolderContent -ChannelId <String> -GroupId <String> -InFile <String>
 [-Data <Stream>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### SetViaIdentity
```
Set-MgGroupTeamChannelFileFolderContent -InputObject <ITeamsIdentity> -InFile <String> [-Data <Stream>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
The content stream, if the item represents a file.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```powershell
Import-Module Microsoft.Graph.Teams
```

Get-MgTeamChannelFileFolder -TeamId $teamId -ChannelId $channelId
