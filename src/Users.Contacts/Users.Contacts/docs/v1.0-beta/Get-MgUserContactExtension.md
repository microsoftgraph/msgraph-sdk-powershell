# Get-MgUserContactExtension

## SYNOPSIS

## SYNTAX

### List (Default)
```
Get-MgUserContactExtension -ContactId <String> -UserId <String> [-Count] [-ExpandProperty <String[]>]
 [-Filter <String>] [-PageSize <Int32>] [-Property <String[]>] [-Search <String>] [-Skip <Int32>]
 [-Sort <String[]>] [<CommonParameters>]
```
